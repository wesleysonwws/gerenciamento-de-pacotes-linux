# Gerenciamento de Pacotes no Linux

## Sobre o projeto

Este repositório demonstra conceitos e práticas relacionadas ao **gerenciamento de pacotes no Linux**, uma atividade essencial na administração de sistemas.

O projeto apresenta comandos e ferramentas utilizadas para **instalação, atualização, remoção e consulta de softwares** em diferentes distribuições Linux.

São abordadas ferramentas utilizadas nas duas principais famílias de distribuições:

- Distribuições baseadas em **Red Hat**
- Distribuições baseadas em **Debian**
